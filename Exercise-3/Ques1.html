Web programming is used to create websites, online stores, web applications, and other online services that are accessed through the Internet.

Web programming languages like HTML, CSS, JavaScript, PHP, and Python are used to create these web-based applications.


Client Side Technologies are the tools and languages that run in the user's web browser, rather than on a web server.

They determine how the content will be displayed to the end-user.

Some popular client-side technologies you might have heard of include HTML, CSS, and JavaScript.


The term 'server-side technologies can encompass a range of software solutions,

mainly: server-side scripting languages;

Database Management Systems (DBMS); web server software such as Apache;

and many other technologies depending upon the application being built.
